XML vs HTML Practice

Task 1: Simple HTML Page

Create a basic HTML page with headings, paragraphs, and a list:

<!DOCTYPE html>
<html>
  <head><title>HTML Example</title></head>
  <body>
    <h1>HTML Example</h1>
    <p>This is a paragraph.</p>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>
  </body>
</html>

Task 2: Simple XML Document

Create a custom XML document to store data about books:

<?xml version="1.0" encoding="UTF-8"?>
<books>
  <book>
    <title>Book One</title>
    <author>Author A</author>
    <year>2023</year>
  </book>
</books>

Task 3: Compare Tag Rules

  • HTML tags are predefined; XML tags are custom.
  • HTML is not case-sensitive; XML is case-sensitive.
  • HTML allows optional closing tags; XML requires all tags closed.

Task 4: Add Attributes

Add attributes to both HTML and XML elements:

<img src="image.jpg" alt="Example Image"> <!-- HTML attribute example -->
<book id="b1"> ... </book> <!-- XML attribute example -->

Task 5: Validate Your XML

  • All tags are properly closed.
  • Tag names match exactly (case-sensitive).
  • No overlapping tags.
